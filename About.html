About Us

Hi, saya adalah web developer yang berdomisili di Yogyakarta. Saat ini sedang belajar HTML di Universitas Mercu Buana Yogyakarta. Saya memang masih baru dalam web development, karena itu saya tidak akan pernah berhenti belajar. Saya ingin menguasai bahasa HTML, CSS, dan Javascript.

Selain itu, saya juga memiliki minat dalam desain antarmuka pengguna (UI) dan pengalaman pengguna (UX).Saya sering membaca artikel-artikel terbaru untuk meningkatkan keterampilan saya dalam hal ini. Sejauh ini, saya telah mengembangkan beberapa proyek kecil, termasuk pembuatan website portfolio pribadi dan beberapa halaman web sederhana.

Selain fokus pada pengembangan web, saya juga tertarik dalam mempelajari teknologi terbaru dalam industri IT, seperti pemrograman aplikasi mobile dan pengembangan perangkat lunak berbasis cloud. Saya percaya bahwa dengan semangat belajar yang kuat dan komitmen untuk terus berkembang, saya akan mencapai tujuan saya untuk menjadi seorang web developer yang terampil dan inovatif.
